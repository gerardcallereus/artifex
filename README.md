# Artífex

Proyecto educativo de marca personal, diseño gráfico y producción de joiería. Una guía completa desde la ideación hasta la comercialización.

## 🚀 Comenzar

### Requisitos previos
- Node.js 18 o superior
- npm o yarn

### Instalación

```bash
npm install
```

### Desarrollo

Para ejecutar el servidor de desarrollo:

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:3000/artifex`

### Construcción

Para crear una versión optimizada:

```bash
npm run build
```

### Preview

Para visualizar la compilación:

```bash
npm run preview
```

## 🛠 Stack tecnológico

- **Astro** - Framework web moderno
- **Starlight** - Tema de documentación para Astro
- **TypeScript** - Tipado estático

## 📝 Fases del proyecto

1. **FASE 1: La guspira** - Ideación inicial
2. **FASE 2: Identitat** - Naming, logotip, colores y tipografia
3. **FASE 3: Disseny Tècnic** - Preparación técnica
4. **FASE 4: Producció** - Fabricación
5. **FASE 5: Costos i preu** - Análisis económico
6. **FASE 6: Comercialització** - Estrategia de venta
